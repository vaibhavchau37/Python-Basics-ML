## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/python-for-ml-practice.git
cd python-for-ml-practice
```

### Install dependencies

```bash
pip install numpy pandas matplotlib seaborn jupyter
```

### Run Jupyter Notebook

```bash
jupyter notebook
```
